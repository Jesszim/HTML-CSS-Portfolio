Admin Dashboard

This is an ultra lightweight admin dashboard based in Blazor. We are experimenting this in our ASP.NET project. The aim is to use Blazor, CSS, CSS Grid and Bootsrap for front end development with the design philosophy of keeping the components modern and lightweight with minimal dependencies. We strongly believe that less dependencies means easier of upgrade. On the other hand, if Blazor or Bootsrap introdues new components that overlap with those in this project, the idea is to let the components in Blazor and Bootsrap take precedence. This is available under the MIT License, so feel free to play with it.
